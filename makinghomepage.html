We also used different ways to make the homepage for our website.

  1. Fireworks
  2. Dreamweaver

Using Fireworks

Pros

  1. It is easy to make and you can make it look exactly how you want it to.
  2. No coding is required.
  3. It is very visual.

Cons

  1. The website you make will only be clickable images, it feels like you're looking at an image of a website rather than an actual website.
  2. Sometimes you might not know what you're clicking on because the "webpage" doesn't react.
  3. It won't look "put together" once you accidentally click and drag on something because you'll realize that the site is actually just made up of images.

Using Dreamweaver

Pros

  1. Dreamweaver is somewhat visual, you can see what you are doing with your website while you edit it in html.
  2. In Dreamweaver, you can use less of coding, you can click on the "Design" tab and edit from there on.
  3. It a better alternative for visual learners rather than just writing code.

Cons

  1. This application is not the typical adobe application people would be used to, it has different functions with different buttons and will take time to learn how to use.
  2. A lot of things on the application are not easy to figure, which means you have to search for the answer if you have a problem instead of finding the solution by yourself playing around on the app.
  3. The application can be confusing to use.
